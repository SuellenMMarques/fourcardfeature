# 🧩 Four Card Feature Section

Projeto desenvolvido a partir de um desafio do Frontend Mentor, com foco em layout responsivo, organização de código e boas práticas de CSS.

## 🚀 Tecnologias utilizadas

- HTML5 semântico
- CSS3
- Flexbox
- CSS Grid
- Mobile First

## 🎯 Objetivo

Construir uma seção de cards responsiva seguindo um layout proposto no Figma, garantindo:

- Estrutura semântica
- Responsividade para diferentes telas
- Organização e reutilização de estilos
- Código limpo e escalável

## 📱 Responsividade

O projeto foi desenvolvido utilizando a abordagem **mobile-first**, adaptando o layout para:

- 📱 Mobile
- 📲 Tablet (≥ 768px)
- 💻 Desktop (≥ 1024px)

## 🧠 Aprendizados

Durante o desenvolvimento deste projeto, foram praticados:

- Uso de **CSS Grid** para layouts complexos
- Organização de estilos com variáveis CSS (`:root`)
- Estruturação semântica com HTML5
- Controle de espaçamento com sistema de design tokens
- Aplicação de responsividade com media queries

## ⚠️ Desafios enfrentados

- Posicionamento dos cards utilizando Grid conforme o layout proposto
- Ajuste de alinhamento dos elementos internos
- Manutenção da consistência visual em diferentes tamanhos de tela
